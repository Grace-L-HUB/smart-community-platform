# 幸福里社区微信小程序

## 项目简介

这是一个为幸福里社区开发的微信小程序，提供业主日常所需的社区服务功能，包括社区公告、物业报修、个人信息管理等。

## 功能模块

1. **首页** - 展示社区概况、最新公告和快速服务入口
2. **公告** - 发布社区通知、活动信息和重要提醒
3. **报修** - 在线提交物业报修申请，上传照片，预约维修时间
4. **个人中心** - 用户信息管理、报修记录查询、缴费信息等

## 开发环境

- 微信开发者工具
- 小程序基础库版本：建议使用最新稳定版

## 项目结构

```
miniapp/
├── miniprogram/
│   ├── pages/                 # 页面文件夹
│   │   ├── index/            # 首页
│   │   ├── announcements/    # 公告页面
│   │   ├── repair/           # 报修页面
│   │   └── profile/          # 个人中心页面
│   ├── images/               # 图片资源文件夹
│   │   ├── *.svg             # SVG图标文件
│   ├── app.js               # 小程序入口文件
│   ├── app.json             # 小程序全局配置
│   └── app.wxss             # 小程序全局样式
└── README.md                # 项目说明文档
```

## 使用说明

1. 克隆项目到本地
2. 使用微信开发者工具导入小程序项目
3. 选择「导入项目」，输入项目名称和项目路径
4. 点击「编译」按钮运行小程序

## 注意事项

1. 小程序使用SVG格式图标，确保微信开发者工具支持SVG格式
2. 当前版本为前端界面原型，部分功能为模拟数据展示
3. 后续需要连接后端API实现完整功能

## 图标资源

当前版本已移除底部导航栏图标配置，因为微信小程序只支持png、jpg、jpeg格式的图标文件。

如果需要添加图标，请将图标文件转换为上述支持的格式，并更新app.json中的iconPath和selectedIconPath配置。

项目中保留了SVG格式的图标文件（home.svg、notice.svg、repair.svg、user.svg等），可以用于参考或转换为支持的格式。
